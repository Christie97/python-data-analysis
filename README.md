# Python Data Analysis Projects

## Overview
This repository contains data analysis projects demonstrating data cleaning,
exploratory data analysis, visualization, and insights using Python and SQL.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Plotly)
- SQL (MySQL, SQLite)
- Jupyter Notebook

## Key Skills Demonstrated
- Data Cleaning & Wrangling
- Exploratory Data Analysis (EDA)
- Data Visualization
- Statistical Analysis
